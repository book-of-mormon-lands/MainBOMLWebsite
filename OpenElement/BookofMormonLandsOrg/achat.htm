Prologue
Senslight les différentes versions
Nous mettons à disposition plusieurs versions du template afin de satisfaire le maximum de personnes.
Pour plus d'informations, vous pouvez consulter le forum.
La version free
- slider sur la home avec utilisation du plugin bxslider 
- nouvel élément appelé Fond d'élément dynamique 100% 
- nouvel élément appelé Panneau d'éléments responsive 
- élément default text 
- élément Parallax sûr 
- élément tooltipster de MicroVinc (pour plus d'infos)
- élément iframe (google map) anti défilement 
- élément Font awesome model 1 de Microvinc (pour plus d'infos)
- élément slideOpener (voir la partie googlemap pour comprendre son utilisé) 
- 3 galeries photos avec des paramétrages spécifiques et l'élément swipebox nouvelle version de Dmit 
- livre d'or de MicroVinc (pour plus d'infos)

Vous pouvez téléchargez la version free sans aucune contrainte ni souci.
Le fichier est au format zip et a été scanné par 2 antivirus. Il est hébergé sur un serveur privé.

Les demandes et retours peuvent se faire directement via cette page du forum.
Pour les différents tutoriels en cours, vous pouvez vous rendre sur cette page du forum.

Lien pour télécharger la version free.
La version complète +  prémium +  éléments
La version complète comprend en plus :
- nouvel élément appelé scrollreveal de MicroVinc (pour plus d'infos)
- disqus pour la partie commentaires des articles (multi langues supportés + tutoriel pour création d'un compte) 
- nouvel élément flexslider de Dmit spécialement développé pour différents projets
- nouvel élément appelé shuffle conterner de Dmit pour une des galeries photos 
- nouvel élément livre d'or de Nico38 modifié spécifiquement pour le template (merci à lui pour sa participation)

Pour bénéficier de la version complète, il est nécessaire d'effectuer un règlement par Paypal.
Par souci d'équité envers les différents intervenants et participants, nous avons jugé important de proposer un prix accessible pour tous.
ceux prenant cette version complète recevront par mail le lien pour récupérer le template ainsi que les différentes procédures (en cours de réalisation) pour certains éléments ou paramétrages comme disqus.

Les demandes et retours peuvent se faire directement via la page du forum ou par mail.
Pour les différents tutoriels en cours, vous pouvez vous rendre sur cette page du forum, certains tutoriels seront envoyés directement par mail.

version premium : 
- support privilégié et mise à jour en avant première par mail.
- aide aux différents réglages pour l'ensemble des éléments pendant 1 mois par la team.
- avant première d'éléments optimisés ou ajout pertinent.

Élément Livre d'or
Nous avons développé un élément spécifique pour un livre d'or avec une gestion sur une page administrateur (login + mdp).
Il est possible de valider et/ou de supprimer un commentaire.
Cet élément peut être acheté à part.

Élément Disqus
Nous avons fait en sorte aussi de développr et d'intégrer un ensemble de code pour la gestion des commentaires avec Disqus.
Il est possible de ne prendre que cette option en acheter que la partie mise en place (une page + 1 article) ainsi que la procédure complète de création pour la suite.

Pour accéder au paiement en ligne, rendez-vous sur cette page.
Nous avons aussi introduit une ligne pour ceux qui prendraient uniquement la version free mais aimeraient apporter leur soutien à tous ceux ayant participés au développement du template. Merci à eux.

Pour toute demande d'informations supplémentaires ou autres, n'hésitez pas à nous contacter par le formulaire ci-dessus.
